service: Museum

Museum

Museum()

Stores and retrieves information from the Museum Route in the API. Used to handle requests made to the API related to the museum, such as:

  • Museum Title
  • Museum Image
  • Hours of Operation
  • Description
  • Social Media Links
  • Location
  • Terms of Service
  • More Info About the Museum
  • Contact Information

Requires Routes and $http services
